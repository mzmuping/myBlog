## 数据库数据类型

### 数值类

|  类型   | 字节  | 最小值(有符号/无符号)  | 最大值(有符号/无符号) |
|  ----  | ----  | --------------------- | ------------------- |
| tinyint   | 1 | -128/0            |        127/255           | 
| SMALLINT  | 2 | -32768/0          |        32767/65535       | 
| MEDIUMINT | 3 | -8388608/0        |       8388607/16777215   | 
| INT/INTEGE| 4 |                   |                          |
| BIGINT    | 8 |                   |                          |

- MySQL中的整型，默认是有符号的。可以通过unsigned 选项设置为无符号(只能表示整数/0)
