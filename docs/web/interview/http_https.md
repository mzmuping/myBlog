## http
http 超文本传输协议,默认80端口，tcp/ip 应用层,明文传输

+ 特点：
    + 无状态（服务器不会记住连接状态，就是之前连接过了，下次还是会重新）
    + 无连接 (一次请求就一次连接，多次就多次连接)
    + 请求与响应状态
+ 过程： 
    + 解析域名
    + dns： 域名解析对应ip地址
    + 网络请求： tcp三次握手 -> 发起http请求 -> http 响应 -> 解析报文，渲染 -> 四次挥手

## https 
在http上加上TSL/ssl 加密数据,

+ 过程
    + http -> ssl/tls加密 -> tcp 传输层 
    + 对称秘钥/非对称秘钥（私密/公密）
        + 对称加密 (运算符or) 
        ```
        //二进制
        10001 原文

        10010 秘钥

        10111 密文
        10010 秘钥
        10001 原文

        ```
    + 摘要算法 （md5,sha2）
    + 数据完整性

http.1 / http.2 区别与优势
