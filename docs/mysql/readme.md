### 创建数据库
    - 创建数据库是在系统磁盘上划分一块区域用于数据库的储存和管理，
      如果管理员在设置权限的时候为用户创建了数据库，则可以直接使用，否则需要自己创建。
    - mysql中创建数据库的SQL语法格式：CREATE DATABASE database_name;
